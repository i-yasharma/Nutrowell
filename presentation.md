# NutroWell: Personalized Student Meal Planner AI
**Presentation & Pitch Guide**

## 🎯 1. The Core Idea
**Problem:** Students struggle to maintain a healthy diet due to tight budgets, erratic class schedules, varied dietary restrictions, and limited access to personalized nutritional advice.
**Solution:** **NutroWell**, a smart web application that generates personalized, practical, and budget-friendly meal plans tailored specifically to a student’s lifestyle, budget, schedule, and health needs.

## ✨ 2. Key Features
- **Holistic Personalization:** Considers fundamental metrics like age, weight, and activity level.
- **Schedule & Budget Integration:** Plugs right into a student's realities—syncs with class timings, travel time, and monthly allowance constraints.
- **Dietary & Health Awareness:** Supports various diets (Vegan, Jain, etc.), food dislikes, allergy warnings, and specific health goals (Exam performance, Immunity, Weight management).
- **Intuitive UI/UX:** A streamlined, step-by-step wizard (About You -> Schedule -> Diet -> Health -> Generate) ensures a frictionless onboarding experience.
- **Instant AI Generation:** Provides a real-time, bespoke meal plan based on the extensive user profile generated in the frontend.

## 💻 3. Technical Implementation
- **Architecture:** Client-side architecture utilizing vanilla HTML, CSS, and JavaScript.
- **Modularity:** Clean separation of concerns with standalone `HTML` (structure), `CSS` (styling, CSS variables, flexbox/grid layouts), and `JS` (user state management, validation, and AI prompt generation/simulation).
- **Responsive Design:** Mobile-first, card-based interface that feels like a native mobile app.
